많은양의 데이터가 실시간으로 전달되어야 할 때 가장 효율적인 방식은 구조화된 텍스트

  • 구조화된 텍스트를 만들기 위해 JSON 데이터를 많이 활용하고 있음
  • 자바스크립트의 객체 형태를 갖고 있기 때문에 구조파악이 쉽고 제작도 어렵지 않다
  • 특히 웹분야에서는 자바스크립트가 모든 프론트 영역에 호환이 된다는 점을 활용하여 데이터 수집 및 출력용으로 거의 모든 플랫폼 서비스에서 사용되고 있음
  • 예를들어 파이썬으로 수집한 데이터를 자바기반 웹 서비스에서 출력하고 싶다면?
  • 파이썬으로 JSON 데이터를 만들어 파일로 자바기반 서버에 전송해주면 자바기반 서버는 웹으로 그 JSON 데이터를 뷰(프론트)영역에 출력시키면 된다
