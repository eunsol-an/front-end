자바스크립트 변수의 특징

변수란? 어느 특정 시점(시간, 공간)에 내가 원하는 값을 갖고 있는 코드(존재)

변수의 특징

  1. 코드가 실행되는 시점의 상태값을 갖고 있는 개념
  2. 변하는 값을 주로 다룬다고 해서 변수라고 칭함
  3. 자바스크립트에서는 변수를 선언(배정, 할당, 정의)할 때 let, var를 사용
  4. ES6(2015) 부터는 let 사용하기를 적극 권장함!
Uncaught SyntaxError: 문법에러

상수 : 한번 선언 및 초기화를 하면 변경할 수 없는 변수

상수명은 대문자로 snake case를 사용함

const 라는 키워드를 사용

상수는 고정된 값을 사용하는 의미로 규칙, 룰, 참조값 등을 정의할 때 사용
