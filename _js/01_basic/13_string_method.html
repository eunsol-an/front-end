문자관련 메서드는 사용률이 매우 높고 다른 프로그래밍 언어에서 사용되는 메서드와 비슷하므로 잘 알아두면 좋다

  • 문자열 부분 추출 관련 메서드 : slice(), substring(), substr()
  • 문자열 대체 메서드 : replace(oldStr, newStr) - 첫번째 매칭되는 문자만 대체, 대소문자 구분
  • 문자열 공백 제거 메서드 : trim() - 데이터 입력값의 좌우 공백을 삭제할 필요가 있는 경우 많이 사용
  • 문자열 추가 메서드 : padStart(목표문자길이, 채울문자), padEnd(목표문자길이, 채울문자) - start는 앞쪽으로, end는 뒤쪽으로 문자열이 추가된다
    문자의 힌트, 전체값 중 일부만 보여주고 싶은 경우 사용
  • 문자 추출 메서드 : charAt(position) - position=index, 지정한 index의 문자값을 추출 / string[index] : readonly
  • 문자열 index 검색 메서드 : indexOf("str"), lastIndexOf("str") - 지정 문자열의 index 리턴, 문자 값이 2자 이상이면 시작하는 char의 index
  • 지정 문자열 검색 메서드 : startsWith(), endsWith() - 지정 문자열이 탐색대상 문자열에서 시작과 끝으로 존재하는지 여부(return true or false)
    파라미터가 1개라면? (지정문자열)의 시작과 끝에 존재하는지 여부
    파라미터가 2개라면? startsWith(지정문자열, 탐색시작 index) / endsWith(지정문자열, 탐색할 문자열의 '시작'부터의 길이
